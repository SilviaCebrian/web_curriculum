# curriculum_web
Proyecto hecho en flutter para utilizarlo de curriculum.

## Compilar web

flutter build web

## Localizacion
- Para añadir los textos ir a lib > l10n > el file en y es y añadir lo que toque.
- Para aplicar los cambios ejecutar flutter gen-l10n.
